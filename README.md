# Ipelets Full Library 
1. This is a collection of all Ipelets from the SoCG media 2024, 2025, and 2026
2. SoCG 2026 corresponds to the files Clustering.lua and NeighborhoodGraphs.lua

## Ipe Installation
1. Go to [The Ipe extensible drawing editor (otfried.org)](https://ipe.otfried.org/)
2. Click on  **`Download the current Ipe version`**

## ▶️ Usage
To use the Ipelets:
- Download the Ipelet file of your choice
- Locate the Ipelets folder on your machine
- Place the downloaded Ipelet file(s) in this folder
- Open **Ipe**.
- Navigate to the **Ipelets** menu.
- Select the desired operation.
  
  <img width="3025" height="3496" alt="Instructions" src="https://github.com/user-attachments/assets/d6875f0c-25d5-4d90-9f54-dbf39e3ff621" />

 

(a) Place a point set in Ipe, (b) select the points, (c) choose a graph from the  ``Ipelets'' menu, (d) enter a parameter value if prompted, (e) the resulting graph is drawn automatically.
